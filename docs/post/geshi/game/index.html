游戏区投稿格式

如果是第一次投稿，可以把说明截图下来哦，这样可以加快你的稿件撰写速度。

在前台投稿时，必须按顺序包含以下内容，如果只包含关键信息也可以，但是审核通过的几率会极小一点，这视投稿内容知名度而定。

文章封面

尽量选择横向长方形的图片，如果没什么需求的尽量不要选择上传视频，如要上传视频也要选择横向长方形画面

标题

请取一个关联性强、言简意赅、方便他人查找的标题

分类目录

如果是游戏的攻略教程等，请选择游戏区-教程

如果是游戏本体，如有露点、性爱等R18内容请选择游戏区-R18，如果不是则选择游戏区-普通

内容

如果是游戏本体

必须按顺序包含以下信息

  • 游戏封面（竖向长方形封面，类似于书籍封面）
  • 该游戏的原文名称
  • 该游戏的中文名称
  • 厂商
  • 发行日期
  • 游戏的百科或介绍地址，也可以替换成你写的游戏介绍
  • 游戏的截图或CG（尽量不要发布露点等色情信息）
  • 游戏的下载链接 （从中国大陆网盘上传的文件请看这里）

如果是游戏攻略教程等

符合规范即可

文章形式

不强求，可根据自己的情况做出选择

  • 默认为第一种
  • 如果封面足够高清请选择第二种
  • 文章包含四个及以上图片可选择第三种

相关文章

可选项，需要输入文章id，文章id就是https://www.vikacg.com/post/后面的数字，如 https://www.vikacg.com/post/233的文章id就是233

若指定，则优先显示指定的相关文章，否则自动生成

标签

如果含有R18内容则必须添加“R18”标签，否则不予通过，其他符合文章即可。

阅读权限

可选项，请参照编辑器页面说明

摘要

可选项，符合文章即可
