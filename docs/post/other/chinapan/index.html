通过位于中国大陆的网盘上传文件

文件注意事项

通过中国大陆的网盘上传文件时，如果含有当地违禁内容，为了您的账户安全，我们建议以下操作：

  1. 将文件打包成为压缩包，以防止被直接抽查
  2. 打包的压缩包最好为分卷文件，防止被在线解压
  3. 压缩包最好带上密码，以防止被直接抽查
  4. 压缩包文件名称请勿包含违禁词汇，最好不要说明是哪种文件的压缩包

请注意，非分卷压缩包更改文件后缀是无用的，文件MD5计算只包含文件内容，会有人在转存文件后通过更改后缀来在线解压。

更优的分享方式（仅限于百度网盘）

什么叫秒传？

用户在使用度盘分享文件，经常会遇到度盘链接被和谐，特别是一些资源站长在分享资源时经常遇到用户反馈度盘分享链接失效了，还得经常更新度盘链接。

PanDownload为此创造了秒传链接，此链接也是一种分享链接，不限文件大小享有一键创建永久有效的特殊分享链接。

作用原理

因为某些原因，pd没了，现在改用标准码生成器。

这个机制是利用了本地计算秒传MD5上传APi来欺骗度盘认为大家都是本地上传了同一个文件已达到分享的原理

  • 说人话就是某些小朋友在线解压只会封他自己号
