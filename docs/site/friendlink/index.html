向维咔申请友链

从今日起，维咔开放友情链接的申请，各大站长可以通过一些简单的手续和维咔交换友情链接，互换流量。

说明：成功申请通过的友情链接将会在以下位置显示：

  1. 非阅读页、分类目录页的页面底部
  2. 绝大多数页面的侧边栏
  3. “导航链接”专页

如要交换友情链接，首先要在您的网站常规或显著位置处添加我们的友情链接，以下为我们的网站信息：

  • 名称：维咔VikACG
  • 图像：https://img0.vikacg.com/wp-content/uploads/2019/07/9be315d4911a4af-192x192.png
  • 链接：https://www.vikacg.com
  • 描述：肥宅们的欢乐家园

然后，在维咔登录您的账户，并前往页面链接专页（https://www.vikacg.com/links）右下角“申请链接”按钮处填写您的网站信息，我们将通过几个维度进行综合分析：

  1. 你方网站是否可通过HTTPS协议访问（支持TSL1.3最佳）
  2. 你方网站是否符合你方服务器所在地法律法规
  3. 我方友情链接是否安装正确
  4. 你方网站的综合情况（视当前申请人数而定，综合指数越优秀的网站越能成功申请）

一般我们会在7个工作日内处理申请，申请成功后双方如无协商，不得撤下对方友情链接，以下情况除外：

  • 某方单方面撤下友情链接
  • 某方网站无法访问
  • 某方运营主体发生变化导致的网页大改版
