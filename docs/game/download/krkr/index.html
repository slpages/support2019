如何使用KrKr模拟器

百科

吉里吉里，又称T Visual Presenter（简称TVP），是一个由W.Dee氏所开发的同人游戏制作工具。它以物件导向语言TJS做为底层开发，外包上KAG的Script语法让一般使用者使用。要使用吉里吉里只需要准备好文本编辑器（如记事本），就可以直接编写剧本。吉里吉里在处理2D图形上极为强大，支持的图像、音乐格式也是当今同人游戏工具中最多的。它让用户可以很方便地整合图像、音乐、动画、文字剧本，用户可以专注在游戏本身而不需要花费太多精神在编写游戏代码上。

吉里吉里的本体采用GPL许可证，使用者不需要负担任何费用即可利用吉里吉里来开发游戏，包括免费、共享或是商业游戏，吉里吉里与NScripter并称为两大同人游戏工具。

下载链接

安卓版下载 | iOS版下载

请注意，不同于 Android , iOS 版本的模拟器需要付费在 App Store 购买。二者都可以在中国大陆区 App Store 进行购买。

你也可以通过其他渠道获得这两款模拟器，但我们始终建议购买正版。

使用教程

安卓版

模拟器目前（也可能是以后）最新的版本是 1.3.9，请务必下载最新版本的模拟器。

下载并打开【吉里吉里2模拟器】，会出现一个文件夹视图。第一次打开时默认的视图从路径 /storage/emulated/0/ （内置储存根目录）开始。在文件夹中找到你把游戏解压到的地方。

在本教程中，解压路径是 /storage/emulated/0/Game ，所以在这里我们找到 Game 文件夹，点击进去，再找到下载好的游戏文件夹。点击【启动游戏.xp3】即可开始游戏。

iOS版

多设备篇（1）——使用数据线

本篇适用于 XP3Player 和 ONSPlayer，且 ONSPlayer 只能通过本方式获得游戏。

除了你手持的 iOS 设备以外，你还需要准备一台电脑（PC/Mac）。暂时没研究出适用于 Linux 的操作方式。 

在电脑上下载游戏并解压，并把所有游戏文件放在同一文件夹。

在电脑上下载并安装爱思助手（官网链接），通过数据线连接设备到电脑。

在左侧的菜单中，找到【应用游戏】选项，点击并在列表中寻找 XP3Player 或 ONSPlayer ，在其右侧找到【浏览】按钮并点击。本教程以 XP3Player 为例演示。

在新出现的页面内选择【导入】➡ 【导入文件夹】，选择刚刚解压好的游戏文件夹。你也可以将整个文件夹直接拖到目录内。

等待导入完成后，就可以打开 iOS 设备上的模拟器，点击【启动游戏.xp3】即可开始游戏，进行游玩了。

多设备篇（2）——使用 WLAN 共享游戏文件

本篇适用于家中有无线网的用户，可以免数据线通过 WLAN 传输游戏文件到iOS 设备。

开始之前请确认你的 iOS 设备和已经有游戏文件的另一台设备（可以是 PC、Mac、Linux 和 Android 手机，iOS 暂未实验）处于同一 WLAN 下，也就是说这两台设备需要连同一个 Wi-Fi 。

在 XP3Player 的设置菜单（右上角的三个横杠）中，找到并打开【Web文件服务器】。打开后，会显示一串内网 IP （即图中的 192.168.43.46）。每个 WIFI 显示的 IP 都不一样，请填入你自己设备给的 IP 地址，不要直接填写教程中的这个 IP。

在你的另外一台有游戏文件的设备上用浏览器打开这个内网 IP 地址，会打开下面的页面。

其中，蓝色按钮指【上传文件】，绿色按钮指【新建文件夹】。

请注意，从这个时候开始请随时留意不要让自己的 iOS 设备息屏！设备息屏后两台设备的网络连接将断开，十分容易造成文件损坏！

由于 XP3Player 的 Web 文件服务器只支持文件传输（不支持文件夹），所以不要直接把整个游戏文件夹拖进去，不然模拟器里面的文件会一团糟，不方便以后其他游戏的载入。

以下是正确的做法：

1.点击绿色按钮创建一个新的文件夹，文件夹以游戏名称为名，方便日后与其他游戏的区分；

2. 点击这个文件夹左边的按钮进入这个文件夹。

3. 查看你游戏文件夹的结构，如果有文件夹，按照同样的办法，在这个目录里新建同名的文件夹。一般而言， KR 版游戏都会有一个 plugin 文件夹，我们在这里新建一个。

2. 将游戏文件夹里除了 plugin 文件夹里的文件全部直接拖到浏览器里（如果是手机操作请使用蓝色的上传按钮）。

在这时应该已经开始共享文件到 iOS 了。等到所有共享操作完成（所有进度条跑满），再打开 plugin 文件夹，如法炮制把 plugin 文件夹里的所有文件直接拖到浏览器里。

回到 iOS 设备，你会发现所有文件都传好了，可以开始游玩了。

单设备篇（慎用）

本篇操作繁琐，十分容易出错，十分不建议新手及急性子尝试。这个方法唯一的优点可能就是不需要其他设备。

在Chrome浏览器下载后选择打开方式为随便一个压缩包解压软件，解压后在iOS自带的文件app从解压软件的目录中复制直接包含游戏文件的文件夹至xp3player的根目录即可

常见问题：

  1. 打开游戏后显示诸如文件配置不完整等信息怎么办？请检查游戏文件根目录是否存在0.txt文件，如不存在请自行添加。
  2. 如果没有【启动游戏.xp3】，则看看目录里有没有【data.xp3】。
  3. 我用吉里吉里模拟器打开游戏的时候显示错误为“cannot find stronge file://./date/app/......”的提示框： 遇到这种问题多半是模拟器版本太低而导致。想要稳定的游玩游戏，请至少升级到 1.3.4 版本。部分旧手机也有可能出现这种问题。
